- Usage -

Include the JS and CSS files from the dist directory. Make sure you include jQuery and jQuery UI first.

<link rel="stylesheet"
      type="text/css"
      href="/path/to/neon-flying-squids.css" />

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
<script src="https://ajax.googleapis.com/ajax/libs/jqueryui/1.12.0/jquery-ui.min.js"></script>
<script src="/path/to/neon-flying-squids.js"></script>

Each element to be animated must have a unique data-squid identifier.

<div data-squid="squidley">
    *waves tentacles*
</div>

Call beget once to initialize the plugin.

$(document).ready(function() {
    nfs.beget();
});

Animate an element by passing its identifier to glideIn, glideOut, or glideToggle, along with the desired direction of animation.

nfs.glideIn('squidley');

nfs.glideOut('squidley');

nfs.glideToggle('squidley');

Example usage: Add a click handler to a button that triggers the glide.

<button class="toggle-squidley">Poke the Squid</button>

$('.toggle-squidley').on('click', function() {
    nfs.glideToggle('squidley');
});
